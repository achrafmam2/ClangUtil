  • removeIncludes(source:)

    Removes include directives from the file (e.g., #include ). Includes are replaced by an empty string.

    Example:

    1. #include <stdio.h>
2. #include <string.h>
3. void main(void) {}

    Will be converted to:

    1.
2.
3. void main(void) {}

    Remark

    Return Value

    Source code without include directives.
